# Deep Multi-Task Learning for SSVEP Detection and Visual Response Mapping

Using multi-task learning to capture signals simultaneously from the fovea efficiently and the neighboring targets in the peripheral vision generate a visual response map. A calibration-free user-independent solution, desirable for clinical diagnostics. A stepping stone for an objective assessment of glaucoma patientsâ€™ visual field.
